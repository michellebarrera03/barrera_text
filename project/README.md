
- Torn parchment paper styled blog cards
- Upvote / Downvote on each post
- Comment section per post
- Pastel gold & candlelight color palette
- Vintage serif typography (Cinzel + IM Fell English)

## File Structure
```
my-blog/
├── img/          ← add your images here
├── index.html    ← main blog page
├── style.css     ← all styling
└── README.md     ← this file
```

## How to Use
1. Upload all files to your GitHub repository
2. Go to **Settings → Pages** and enable GitHub Pages
3. Your blog will be live at `https://yourusername.github.io/your-repo-name`
